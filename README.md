# Projeto PIM II — Sistema de Gestão Educacional

Este repositório contém o projeto PIM II, desenvolvido para gerenciar informações acadêmicas de alunos, professores, turmas, notas e disciplinas.  
O sistema utiliza as linguagens Python e C, além de modelos Astah (.asta), para representar a estrutura e o funcionamento do sistema de forma modular e organizada.

---

## Estrutura do Projeto

Projeto-PIM-II/
│
├── src/  
│   Código-fonte principal  
│   ├── Nexus.py — Script principal em Python  
│   ├── pim.c — Código em linguagem C

│   ├── banco de dados.c — Implementação do banco de dados

└── README.md — Documentação do projeto

## Tecnologias Utilizadas

- Python 3.11 ou superior  
- Linguagem C  
- Astah UML  
- Arquivos CSV  
- Git e GitHub  

---

## Funcionalidades

- Cadastro e gerenciamento de alunos, professores e turmas  
- Registro e cálculo de notas  
- Leitura e gravação de dados em arquivos CSV  
- Integração entre módulos em Python e C  
- Modelagem do sistema através de diagramas UML
